# Legacy Archive (2026-04)

本目录存放本轮迁移归档的过时数据，保留原结构用于回溯与面试材料追踪。

## 迁移来源

- 旧后训练样本路径：`data/mock/**`（已退出主流程）
- 历史样本集：`data/cases/**`、`data/environments/**`、`data/rl/**`

## 迁移原因

1. 统一新口径为 `data/eval_samples`。
2. 历史目录字段契约与新评测字段不一致，不再作为主评测入口。
3. 采用“迁移归档”而非删除，确保可追溯。

## 回溯方式

- 需要历史样本时，直接从本目录按原路径读取。
- 新评测仅使用 `data/eval_samples`。
